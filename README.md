# Weather-App

🌤️ Weather Forecast App

A clean and responsive weather application that displays real-time weather data for cities across the United Kingdom.

🚀 Features


🔍 City Search — Search any UK city and get instant weather data
📍 Geolocation — Detect your current location automatically
🌡️ Unit Toggle — Switch between °C and °F
🌙 Dark / Light Mode — Toggle between themes
🕘 Recent Searches — Quickly access your last searched cities
🇬🇧 UK Focused — Pre-loaded with all 76 official UK cities as suggestions
🏳️ Country Flag — Displays the country flag alongside the city name

 Technologies Used


HTML5
SCSS (modular with variables)
Vanilla JavaScript (ES6 Modules)
OpenWeatherMap API
Bootstrap Icons
FlagCDN (for country flags)
